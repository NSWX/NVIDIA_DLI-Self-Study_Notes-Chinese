## 🎯 什么是 NVIDIA DLI？

NVIDIA 深度学习学院（**D**eep **L**earning **I**nstitute，简称 DLI）提供由 NVIDIA 专家设计的实践培训课程，帮助开发者和IT专业人员掌握 GPU 加速计算、AI 和深度学习的核心技能。

### 📚 课程类型说明

| 课程类型                | 说明                       | 图标 |
| ----------------------- | -------------------------- | ---- |
| **NVIDIA 能力认证课程** | 🎓 完成后可获得官方能力证书 | ⭐    |
| **自助学习课程**        | 📖 随时随地自主学习         | 📖    |
| **讲师授课工作坊**      | 👨‍🏫 专业讲师实时指导        | 🎯    |
| **认证考试**            | 📝 获得官方认证资质         | 🏆    |
